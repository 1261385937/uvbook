Glossary
========

.. glossary::

    `libuv <https://github.com/joyent/libuv>`_
        A cross platform abstraction to provide a high performance event loop
        on Windows and UNIX. libuv wraps IOCP on windows and libev on Unix. It
        was originally designed for the :term:`node.js` project.

    `node.js <http://nodejs.org>`_
        Evented I/O for JavaScript

    watcher
        libuv structure encapsulating data about a certain kind of event.

    handle
        See :term:`watcher`
